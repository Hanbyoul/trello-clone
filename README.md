# TrelloClone

> Trello의 drag and drop 구현프로젝트입니다.

### 배포

https://gentle-bienenstitch-3f2559.netlify.app/

## 프로젝트 목적

- react-beautiful-dnd Drag&Drop 구현
- react-recoil 상태 관리
- LocalStorage data 저장
- react-Memo 성능 최적화
